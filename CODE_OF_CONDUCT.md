# Code of Conduct

## Our pledge

We want this project to be a welcoming, harassment-free community for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent; ask clarifying questions.
- Keep discussions focused on whatâ€™s best for users of this project.
- Respect privacy and security. Do not share private calendar data, API keys, or sensitive logs.

## Unacceptable behavior

- Harassment, hate speech, or discrimination.
- Threats, doxxing, or sharing personal information without consent.
- Sexualized language or imagery.
- Deliberate disruption (spam, trolling, repeated bad-faith arguments).

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the maintainers.

- Preferred: GitHub Security Advisories (private report)
- Otherwise: email the maintainer address listed in the repository profile

Include:
- What happened (links, screenshots, timestamps)
- Who was involved
- Any context that helps us understand the situation

## Enforcement

Maintainers may remove content, block accounts, or take other actions they deem appropriate to keep the community safe.

## Scope

This Code of Conduct applies in project spaces (issues, PRs, discussions, code reviews) and when representing the project in public.
